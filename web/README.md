This is the Next.js front-end for the AI Code Review project.

## Getting Started

Install dependencies from the `web/` directory, then run the app there:

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view the landing page.

If you are also working on the terminal app in the repository root, keep it in a separate terminal so both apps can run independently.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## Learn More

This app uses Next.js App Router, Framer Motion, and Tailwind CSS. The main entry point is [app/page.tsx](app/page.tsx).
